Group Permission System

FUDforum 2 supports a highly capable permissions system, which can be used to control what forum users can and cannot do. Using this system you can grant special access rights to certain users, block users from seeing and/or accessing certain forum etc...

This system also includes support for "group managers" have the ability to manage users in the groups they control. This option allows the admin to delegate responsibility to trustworthy forum members.

The Group Management is done via two control panels, the administrative group manager, which allows the admin(s) to add/remove groups as well as assign group managers and specify what permissions the group managers can assign and/or take away. This control panel also allows the admin to make groups inherit permissions from other groups, thus not needing to specify new permission every time for similar groups.

The second control panel, the user group manager, which allows the group manager(s) and the admin(s) to add and remove users from a group as well as change the permissions of the users inside the group can be a accessed via a link available to all Administrators and Group Managers on most forum pages.
