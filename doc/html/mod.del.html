Delete Threads/Messages

Deleting a thread

Deletion of a thread is an action, which can be performed by privileged users by clicking on the delete link beside each thread on the thread listing, or by clicking on the delete button of the root (1st message in the thread) message.

Deleting a message

Privileged users can delete a message by clicking on the delete button located at the bottom of each message, that is located in a forum where they have delete privileges.
