Language Selection

This option allows you to change the language your forum is in. The supported languages are listed in the 'Language' select box.

Locale

Each language has a related locale value, the locale setting tells PHP to print all dynamically generated information such as dates in the selected language. This value is filled in automatically for you when you choose a language, if for whatever reason you believe the value is invalid or have a more appropriate locale value you can alter the default by entering a different value inside the 'Locale' textbox.

If you are unsure of what this option does, it is better to leave the default value as the locale setting.

Note

On most *NIX computers you can see the list supported locales by executing this command: locale -a
