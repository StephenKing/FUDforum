Split Thread

Split thread link, which is visible to privileged users when they are viewing the content of the thread allows for splitting of a thread in to two separate threads. This is mostly done to allow the forum staff to move off topic discussions that have wondered in to a relevant thread.

When clicked, the link would take the privileged user to a control form where how the thread will be split can be specified.

At the top of this form there is a textbox where the name of the new thread will be specified as well as a drop down box with a list of forums where the new thread will be placed. Below these controls there are two buttons allowing the privileged user to split the selected or the non-selected messages.

All the messages in the thread are listed on the 'Split Thread Control Panel', the messages can be selected by marking the checkbox, which is located to left of every message.

Clicking on the 'Split selected messages' button would then split all the selected messages or if the 'Split non-selected messages' button is clicked then all the non-selected messages will be moved to a new thread.
