Action Log Viewer

All administrative actions of privileged users are logged by FUDforum 2.3 so that they can later be reviewed by the admin.

This control panel allows the admin to see this information by displaying the user name, action, the affected forum data as well as the time of the event.

Tip

In addition to logging actions of privileged users this control panel also allows the admin to see failed logins attempts into any of the accounts that have administrative access to the forum, with the IP address of the user who attempted to login. This is an extremely useful feature for tracking down attempted hacking attempts, by people who try to gain administrative access by guessing the password of the administrator(s).
