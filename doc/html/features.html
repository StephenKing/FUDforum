FUDforum 2 includes numerous features, this section will briefly overview most of those features to give you an idea about the full capabilities of FUDforum 2.

Spell Checker

FUDforum 2 supports a fully functional spell checker, that if enabled would allow forum users to spell check their messages before posting.

The spell checker utilizes PHP's Pspell library and supports multiple languages.
