Polls

FUDforum 2 supports a complete polling system, which allows users, if permitted to add polls to their messages.

Polls are quick multiple choice questioners, which can be used to gather user options on a matter at hand. The forum tracks the votes and ensures that people cannot vote more then once to prevent ballot stuffing.
