Preface

Abstract

FUDforum 2 is web based forum software designed in PHP, utilizing MySQL or PostgreSQL back-end released under the GPL Open Source license. It is designed to work on any web server that supports PHP, which makes it usable on virtually very OS. The forum includes i18n and templating support allowing for a complete customization of it's output as well as a very capably group based permission system.

This manual is primarily intended towards forum administrators, it's aim to explain how to use the various features of the forum.

This manual is available in several formats, which are downloadable from http://fud.prohost.org/doc/.
