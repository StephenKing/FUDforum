Deleted Forums

When a forum is deleted it is not immediately removed from the database but rather moved to a trash bin of sorts. It is done to allow the admin to restore the accidentally removed forums, since once they are truly removed there is no going back.

This control panel allows the admin to see all the forums inside the 'trash bin' and allows the admin to either restore the forum, by choosing a category where to restore the forum by clicking on the 'Reassign' button or to truly delete the forum by clicking on the delete link.

Tip

It is a good idea to run the Compact Messages utility after deletion of a forum, especially if that forum had many messages in it.
