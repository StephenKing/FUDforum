Forum Speed

Although not a feature per say, it should be noted that despite the many features FUDforum 2 is still extremely fast, probably faster then all the other forums that offer similar and even lesser feature sets. This is accomplished by a very highly tuned MySQL or PostgreSQL interface as well as a very optimized PHP code.

The forum will in most cases not take more then 0.1 seconds to generate a page.
