Mass Mail System

This control panel allows the administrator to send an e-mail message to all the registered users.

Ignore User Override

The user can specify that they wish to ignore admin's emails via their profile settings. However, if the admin feels that the message is important enough that all forum members should receive it, then they can overwrite the user's setting, by selecting this option.
